# MCP Integration (Removed)

`orcheo_sdk.mcp_server` and the `orcheo-mcp` CLI entrypoint were removed from active Orcheo runtime/tooling as part of Python-only workflow composition.

If you need historical reference material, see [legacy/docs/mcp_integration.md](../legacy/docs/mcp_integration.md) in this repository.

For automation and agent-facing workflow operations, use the CLI and Agent Skills flows instead.
